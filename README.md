# Post React Redux

## Description

A posts app using React and Redux-Toolkit

## Feature

- Vite v3
- React
- Eslint
- Prettier
- Tailwidn CSS v3

## Getting started

```
npm install
```

```
npm run dev
// Open localhost:3000
```

## Extra

Install the package json-server to start the local server

```
npm i json-server -g
json-server --watch data/db.json --port 3500
```
